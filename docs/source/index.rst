What is PhyCom?
---------------

The physical computing bridge (PhyCom bridge) provides a platform from 
hardware as well as software side environment to achieve physical computing scope. 

PhyCom was developed at `CEDT,NSUT <http://cedtnsit.in/>`_.
Phycom kit is being manufactured and marketed by `SincGrid LLP <http://sincgrid.in/>`_.

Technical aspect
----------------

PhyCom is platform which have server side implementation of firmata, 
client-server (request-response) model in its MCU. 
This PhyCom is controlled by a host computer.  
This host computer acts as client in this client-server model.
